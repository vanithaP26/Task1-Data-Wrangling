# Data Cleaning & Wrangling Project

## Objective
To clean and prepare raw dataset for analysis.

## Steps Performed
- Loaded dataset using pandas
- Identified missing values and duplicates
- Handled missing data
- Removed unnecessary columns
- Converted date format
- Removed duplicates

## Tools Used
- Python
- Pandas

## Output
Cleaned dataset stored in output/cleaned_data.csv

## Dataset

The dataset used in this project is sourced from Kaggle.

Dataset Name: Unlock Profits with E-Commerce Sales Data  
Source:https://www.kaggle.com/datasets/thedevastator/unlock-profits-with-e-commerce-sales-data
